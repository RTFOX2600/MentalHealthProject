# 思想关怀系统

# 项目简介

这是一个用通过分析各种数据、进行用户思想状态分析的系统。
可以辅助思想健康分析工作、精准思政工作、精准扶贫工作、舆论监控工作等。

各个部门以及辅导员上传食堂消费统计数据、门禁记录数据、校园网记录数据等，
系统会定时对数据进行处理、分析，并生成每个年级、学院、专业的报告，
同时提供简单的筛选、排序、统计等功能，辅助辅导员进行思想状态分析工作。

# 项目技术栈
### 核心（必须）

**Django：主框架**

Django + Django 模板 + Django Admin + 简单 JS

适合本项目的场景：

- 内部管理系统
- 以数据展示和操作为主
- 对界面交互要求不是极端复杂
- 希望快速开发上线

优势：

- 开发速度快：Admin 后台秒生成
- 维护简单：前后端一体，部署方便
- 响应式：可用 Bootstrap 等实现移动适配
- SEO 友好：服务端渲染
- 成本低：一人可全栈开发（本项目目前 2 人）

**Bootstrap：响应式 UI**

Bootstrap 是一个开源的前端框架，用于快速构建响应式、移动优先的网站和 Web 应用。它提供了一套预定义的 CSS 样式、组件（如导航栏、按钮、表单、卡片等）和 JavaScript 插件，开发者可以直接使用这些样式和组件，无需从零编写 CSS。

**Chart.js：简单图表**

Chart.js 是一个轻量级的 JavaScript 图表库，用于在网页上绘制各种统计图表（折线图、柱状图、饼图等）。它基于 Canvas 实现，使用简单，只需传入数据即可生成图表。

**DataTables：表格功能**

DataTables 是一个 jQuery 插件，用于增强 HTML 表格的功能，提供排序、搜索、分页、导出等高级功能。它可以将普通的 HTML 表格变成功能丰富的交互式表格。

**PostgreSQL：数据库**

- 开发阶段：继续使用 SQLite，简单方便
- 上线前：切换到 PostgreSQL（性能更好，并发支持强）
- 切换方法：修改 Django 的`settings.py`中的`DATABASES`配置即可，代码无需改动（Django ORM 屏蔽了数据库差异）
