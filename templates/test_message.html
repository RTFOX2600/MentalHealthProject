使用文档

消息提示 API

window.showMessage(message, type, duration) 参数说明： - message: 消息内容（字符串） - type: 消息类型 'info' | 'success' | 'warning' | 'error' - duration: 显示时长（毫秒），0 表示不自动消失 返回值：消息ID（可用于手动关闭） 示例： window.showMessage('操作成功！', 'success', 3000); const msgId = window.showMessage('加载中...', 'info', 0); window.closeMessage(msgId); // 手动关闭

确认对话框 API

window.showConfirmDialog(options) 参数说明： options = { title: '对话框标题', message: '对话框内容', primaryText: '确认按钮文字', secondaryText: '取消按钮文字', onPrimary: function() { /* 确认回调 */ }, onSecondary: function() { /* 取消回调 */ } } 示例： window.showConfirmDialog({ title: '删除确认', message: '确定要删除这条记录吗？此操作不可恢复。', primaryText: '确认删除', secondaryText: '取消', onPrimary: function() { window.showMessage('已删除', 'success', 3000); } });
